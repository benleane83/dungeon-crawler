# Squad Team

## Project Context

- **Project:** Browser-based roguelike dungeon crawler
- **Stack:** HTML5 Canvas, vanilla JavaScript, LocalStorage
- **User:** Ben Leane
- **Description:** Procedurally generated dungeons, permadeath, turn-based combat. 10 floors, 3 character classes, fog of war, item identification system.
