## Integration points
- Combat calls `generateLootDrop()` on enemy death
- Dungeon calls `generateItem()` during `spawnItems` hook
- Renderer calls `getDisplayName()` for all item display
- Game loop calls `initIdentificationSystem()` at game start
- Save/load calls `serializeIdentificationState()` / `restoreIdentificationState()`

## Key design
Module-level identification state persists across session. Equipment bonuses mutate Entity stats directly.
